    exploring_mars v0.1.0 API Reference

    Modules

    ExploringMars.CLI

    Top-level module for the CLI application, encapsulating the interpretation of command-line arguments. Delegates actual execution to the MissionRunner module.

    ExploringMars.Mission

    This module defines a mission. A mission encapsulates a single run of a probe, being defined by a single set of problem parameters (bounds, initial position, and instructions).

    ExploringMars.Mission.Coordinate

    This module defines functions that create and operate on coordinates in the probe's space of movement.

    ExploringMars.Mission.Direction

    This module defines functions that create and operate on directions in the probe's coordinate space.

    ExploringMars.Mission.Instruction

    This module defines functions that operate on the probe's instructions.

    ExploringMars.Mission.Position

    This module defines functions that operate on the probe's position.

    ExploringMars.MissionRunner

    File I/O handling module. Reads parameters from an io_device (which should be either a file or :stdio and runs each mission using the Mission module.
