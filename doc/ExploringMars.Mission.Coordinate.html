    exploring_mars v0.1.0 ExploringMars.Mission.Coordinate

    This module defines functions that create and operate on coordinates in the probe's space of movement.

    This module and the Direction module should change if the coordinate representation used in the problem changes - for instance, if we choose to use {y, x} coordinates instead of {x, y} or if we choose to model a probe moving in 3D space.

    This means this module is coupled to the Direction model, as it depends directly on the representation of the Direction.t type. This is acceptable for now, as the Direction.t type is quite simple, but it might need change. Check the note in the documentation of the Direction module for further details.
